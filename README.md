### 啥是App store EPP

员工购买计划 (“计划”) 是 Apple Inc. (“Apple”) 提供的优惠，它适用于参与该计划的公司和其他组织 (“雇主”) 的在职员工、劳务合同工、会员及家庭成员。本计划旨在供个人自用或朋友和家人使用。
